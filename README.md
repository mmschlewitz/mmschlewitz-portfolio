# Melissa Meixner-Schlewitz - Portfolio Website

A modern, responsive portfolio website showcasing my UX/UI design work, skills, and experience.

## Overview

This portfolio website features:
- Clean, professional design with smooth animations
- Fully responsive layout (mobile, tablet, desktop)
- Accessible and WCAG 2.1 compliant
- Optimized performance and SEO
- Social media sharing integration (Open Graph, Twitter Cards)

## Features

- **Hero Section**: Introduction with call-to-action buttons
- **Stats**: Key metrics and achievements
- **Projects Gallery**: Featured case studies with detailed descriptions
- **Skills Section**: Comprehensive list of tools and methodologies
- **Credentials**: Education, certifications, and experience
- **Contact Form**: Easy way to get in touch
- **Smooth Navigation**: Mobile-friendly menu with smooth scrolling

## Technologies Used

- HTML5
- CSS3 (Custom Properties, Flexbox, Grid)
- Vanilla JavaScript (ES6+)
- Responsive Design
- Intersection Observer API for animations

## Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/mel-portfolio.git
   ```

2. Open `index.html` in your browser or serve with a local server:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js
   npx http-server
   ```

3. Visit `http://localhost:8000` in your browser

## Customization

### Adding Images

Replace placeholder images in the `images/` directory:
- `profile.jpg` - Your professional headshot
- `project-1.jpg` through `project-4.jpg` - Project screenshots
- `favicon-32x32.png`, `favicon-16x16.png` - Favicons
- `apple-touch-icon.png` - Apple touch icon
- `og-image.jpg` - Open Graph image for social sharing (1200x630px recommended)
- `twitter-image.jpg` - Twitter card image (1200x600px recommended)

### Updating Content

Edit `index.html` to update:
- Personal information and bio
- Project descriptions and links
- Skills and expertise
- Credentials and certifications
- Contact information

### Styling

Modify `css/styles.css` to customize:
- Color scheme (CSS custom properties at the top of the file)
- Typography
- Layout and spacing
- Animations and transitions
